# Task Management Web Application

A simple web-based task management system built using Python Flask and HTML.
This project helps users add, view, and manage tasks efficiently.

## Features
- Add new tasks
- View task list
- Persistent storage using SQLite
- Simple and clean UI

## Technologies Used
- Python (Flask)
- HTML & CSS
- SQLite
- Git & GitHub

## How to Run
1. Install Flask:
   pip install flask
2. Run the application:
   python app.py
3. Open browser and go to:
   http://127.0.0.1:5000/
